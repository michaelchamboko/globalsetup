# Frontend Designer & Accessibility Reviewer Profile

Review changes for visual alignment, responsive design, user experience, and accessibility standards (WCAG).

## Checklist

- [ ] **Responsive Display**: Layout renders correctly on mobile, tablet, and desktop viewports.
- [ ] **Design Tokens**: Colors, margins, padding, and fonts use existing design tokens. No hardcoded colors.
- [ ] **Keyboard Navigation**: Interactive elements (buttons, links, dropdowns) can be accessed using only the Tab and Enter keys.
- [ ] **Focus Indication**: Focused elements have a distinct, high-contrast visual indicator.
- [ ] **Form Accessibility**: All input fields have associated <label> tags or ria-label attributes.
- [ ] **Contrast Ratio**: Text contrast satisfies the WCAG AA requirement (minimum 4.5:1 for normal text).
- [ ] **Touch Usability**: Buttons and touch targets have a minimum size of 44x44px.
