# Product Requirements Reviewer Profile

Review changes for alignment with product goals, requirements mapping, and user workflows.

## Checklist

- [ ] **Traceability**: Every requirement ID (R-ID) from the PRD is mapped to completed code implementation.
- [ ] **Feature Scope**: The feature matches the build brief and PRD. No scope creep or unrequested features.
- [ ] **User Experience**: The user flows match the description in the target user stories.
- [ ] **Success Metrics**: The code provides the necessary instrumentation (e.g. logs, events) to track success metrics.
